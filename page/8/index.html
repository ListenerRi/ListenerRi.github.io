linux主要目录的作用

发表于 2016-04-22 更新于 2019-08-08 分类于 linux 阅读次数： Valine：

手动敲一遍、算是加强记忆吧~

/：文件系统的入口，也是最高一级的目录

/bin：最基本的且着急用户和普通用户都可以使用的命令放在此目录下，如：ls、cp等

/boot：存放Linux的内核及引导系统所需要的文件，包括引导装载程序

/etc：存放系统配置文件，一些服务器的配置文件也放在这里

/dev：存放所有的设备文件，比如声卡、磁盘等
